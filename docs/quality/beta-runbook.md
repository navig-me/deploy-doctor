# Real-World Beta Runbook (10 Repos)

## Goal
Run deploy-doctor against 10 external repos and capture quality signals.

## Per-Repo Checklist
- Repo URL
- Stack type
- Profile selected
- Findings count by severity
- False positives noted
- Actionability rating (1-5)

## Output Template
Use `docs/quality/beta-report-template.md`.
